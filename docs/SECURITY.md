# Security Policy

## Supported Versions

Security fixes are applied to the current `main` branch.

## Reporting a Vulnerability

Please report vulnerabilities at <https://github.com/favonia/evfmt/security/advisories/new>

## Handling a Vulnerability

The maintainer, [favonia](mailto:favonia+github@gmail.com), will respond to your report as soon as possible and potentially work with you to fix the issue.
